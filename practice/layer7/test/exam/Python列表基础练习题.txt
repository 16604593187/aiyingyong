### 题目1：创建与访问列表
创建一个包含 5 个你最喜欢的水果的列表，然后分别打印出列表中的第一个元素和最后一个元素。

**答案：**
```python
fruits = ['苹果', '香蕉', '草莓', '橙子', '葡萄']
print(fruits[0])   # 苹果
print(fruits[-1])  # 葡萄
```

### 题目2：列表切片
给定列表 `nums = [10, 20, 30, 40, 50, 60, 70]`，请用切片操作获取以下子列表：
1. 前三个元素
2. 从索引2到索引5（不包含5）的元素
3. 所有偶数索引位置的元素

**答案：**
```python
nums = [10, 20, 30, 40, 50, 60, 70]
print(nums[:3])      # [10, 20, 30]
print(nums[2:5])     # [30, 40, 50]
print(nums[::2])     # [10, 30, 50, 70]
```

### 题目3：列表方法操作
已知列表 `scores = [88, 92, 75, 88, 90, 88]`，请完成以下操作：
1. 在末尾添加一个新分数 95
2. 统计 88 出现的次数
3. 找到 75 的索引位置
4. 对列表进行升序排序

**答案：**
```python
scores = [88, 92, 75, 88, 90, 88]
scores.append(95)           # 添加 95
print(scores.count(88))     # 3
print(scores.index(75))     # 2
scores.sort()               # 升序排序
print(scores)               # [75, 88, 88, 88, 90, 92, 95]
```

### 题目4：列表遍历与修改
有一个列表 `numbers = [1, 2, 3, 4, 5]`，请将列表中的每个元素都乘以 2，生成新的列表，并打印结果。

**答案：**
```python
numbers = [1, 2, 3, 4, 5]
doubled = [n * 2 for n in numbers]   # 列表推导式
print(doubled)  # [2, 4, 6, 8, 10]

# 或者使用普通 for 循环：
doubled2 = []
for n in numbers:
    doubled2.append(n * 2)
print(doubled2)  # [2, 4, 6, 8, 10]
```

### 题目5：列表合并与去重
有两个列表 `list_a = [1, 2, 3, 4]` 和 `list_b = [3, 4, 5, 6]`，请：
1. 合并两个列表
2. 去除合并后列表中的重复元素
3. 最终结果按升序排列

**答案：**
```python
list_a = [1, 2, 3, 4]
list_b = [3, 4, 5, 6]

merged = list_a + list_b               # 合并
unique = list(set(merged))             # 去重
unique.sort()                          # 排序
print(unique)  # [1, 2, 3, 4, 5, 6]

# 一行完成：
result = sorted(set(list_a + list_b))
print(result)  # [1, 2, 3, 4, 5, 6]
```
